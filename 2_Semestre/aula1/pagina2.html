Introdução sobre HTML

A HTML é ma linguagem de Marcação de HiperTexto, ela informa ao navegador que um texto deve ser apresentado de uma forma diferente, como por exemplo;

  1. Parágrafo
  2. tabelas
  3. Imagem
  4. Vídeo
  5. Links

Para fazer as marcações utilizamos tag (etiquetas) que indicam o início e o fim de uma marca (elemento)

Principais tags

tag Objetivo Exemplo
p Cria um Parágrafo

Paragrafo
br cria uma quebra de linha Palavra 1
Palavra2
h1 cria um título

Título da página
a cria um link Link
input cria um campo de formulário
button cria um botão
table cria uma tabela
tr cria uma linha na tabela
td cria uma célula na tabela
video insere um vídeo na página
img insere uma imagem na página
